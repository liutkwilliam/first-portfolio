The full stack web app project is completed as part of a course work in web application development in May 2025 with the University of Sydney

The project is completed between 4 group members of studying computer science, software engineering and interaction design. Completed with Jing, Sharang and Gillian.

The project required team members to design and develop a full-stack web application for a second-hand phone online purchasing system, along with an administrative system, using the MERN stack (MongoDB, Express.js, React.js, and Node.js), based on the project brief. The design was modeled after commercial online retail platforms to ensure real-world scenarios and edge cases were considered.

The databases are connected and accessed through asynchronous communication, allowing data to persist within the web application unless updates are made.
