William designed and built his personalised website portfolio to better showcase his experiences in website development. The website is built with GitHub Pages and Jekyll Static Scripting Code and written with basic HTML, CSS and some javascript.

Before this website, William had a portfolio website on a third-party website builder, he found the website builder didn't have enough flexibility to customise layout design with convenience and choices. he decided to build his own one to fulfil his passion and solve the struggle.

In late December, he came out with his personalised logo that would represent his name and himself as a designer, passionate in 2D animation, photography, graphic design and website development.

A recent update was made in Dec 2020.
